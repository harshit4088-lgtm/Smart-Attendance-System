# 📊 Smart Attendance System

A modern and simple **web-based attendance management system** built using HTML, CSS, and JavaScript.
This project allows admins to manage student records, mark attendance, and view profiles in a clean dashboard UI.

---

## 🚀 Features

* 🔐 **Admin Login System**
* ➕ **Add Student Records**
* 📅 **Daily Attendance (Present / Absent)**
* 👤 **Student Profile Popup**
* 💾 **LocalStorage Data Save**
* 📱 **Responsive Design (Mobile Friendly)**
* 🗑️ **Delete Student Option**
